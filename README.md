# Fuex
This app merge excel by mapping columns of a main excel to columns of another excel.
It use python with pandas and tkinter.

## How to start ?
Requirements :
```
pip install -r requirements.txt
```

Start the app :
```
python app.py
```




**Or you can just start the app with Fuex.exe (in dist/fuex.exe)**
